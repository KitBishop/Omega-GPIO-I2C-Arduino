# **iotemplate**
A template for a C++ program that can be used as the basis for user written code that makes use of libnewgpio and/or libnewi2c and/or libarduino libraries on the Omega

See file **`iotemplate.pdf`** for detailed documentation
See also files **`libnewgpio.pdf`**, **`libnewi2c.pdf`** and **`libarduino.pdf`** for detailed documentation on the libraries
